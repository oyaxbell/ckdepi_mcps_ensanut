# Comparison of Low eGFR Prevalence and Prediction for Mortality Using 2009 and 2021 CKD Epidemiology Collaboration Equations in Mexican Adults

Code and Supplementary data for paper: "Comparison of Low eGFR Prevalence and Prediction for Mortality Using 2009 and 2021 CKD Epidemiology Collaboration Equations in Mexican Adults"

Link to paper: https://journals.lww.com/cjasn/abstract/9900/comparison_of_low_egfr_prevalence_and_prediction.813.aspx

DATA AVAILABILITY: Data from the Mexico City Prospective Study are available to bona fide researchers. For more details, the study’s Data and Sample Sharing policy may be downloaded (in English or Spanish) from https://www.ctsu.ox.ac.uk/research/mcps. Available study data can be examined in detail through the study’s Data Showcase, available at https://datashare.ndph.ox.ac.uk/mexico/.

Data from ENSANUT is available here: https://ensanut.insp.mx/
